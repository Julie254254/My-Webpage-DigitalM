# Cia Juliet Portfolio

A modern, minimal, and chic digital marketing portfolio website built with HTML5, CSS3, and JavaScript.

## Features

- Responsive design (mobile-friendly)
- About, Home, and Contact pages
- External profile image and LinkedIn integration
- Minimalist color palette (pink & rose gold)
- Testimonials and skills section
- Contact form with validation (JavaScript)
- Deployed with GitHub Pages

3. Edit content as needed for your own portfolio.
